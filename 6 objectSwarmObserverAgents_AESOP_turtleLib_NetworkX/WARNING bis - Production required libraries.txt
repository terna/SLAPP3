To run the ‘production’ application it is necessary to install

matplotlib which is at http://matplotlib.org

and

NetworkX, which is at http://networkx.github.io


To run the 'oligopoly' application (it is at https://github.com/terna/oligopoly)
and, more generally, to use a database tool within SLAPP,

it is necessary to install
pandas which is at http://pandas.pydata.org

NB
NetworkX and matplotlib versions specified in file runningSpecificCases.md into oligopoly project
folder
